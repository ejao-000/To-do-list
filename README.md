To-Do List App

A clean, notebook-styled to-do list app built with pure HTML, CSS, and JavaScript. No frameworks, no installs — just open and use.

---

## 📸 Preview

A spiral notebook-themed task manager with smooth animations, task filtering, and automatic saving.

---

## ✨ Features

- **Add tasks** — Type and press Enter or click the `+` button
- **Tag tasks** — Label them as `work`, `personal`, `study`, or `errand`
- **Complete tasks** — Click the circle checkbox to mark as done
- **Delete tasks** — Hover over a task to reveal the ✕ button
- **Filter tasks** — Switch between All, Active, and Done views
- **Progress bar** — Fills up as you complete tasks
- **Live stats** — See total, completed, and remaining task counts
- **Clear completed** — Remove all done tasks in one click
- **Auto-save** — Tasks are saved in the browser using `localStorage`

---

## 🚀 Getting Started

### Prerequisites
No installs needed! Just a web browser (Chrome, Firefox, Safari, Edge).

### Running the App

1. Download the `todo.html` file
2. Double-click it to open in your browser

That's it — the app is ready to use! ✅

## 📁 Project Structure

This is a single-file project:

```
todo.html
├── HTML → Page structure (notebook layout, inputs, task list)
├── CSS → Styling (paper/notebook theme, animations, hover effects)
└── JavaScript → Logic (add, complete, delete, filter, save tasks)
```
